TASK 7

Завдання для роботи
1. Напишіть програму, яка формує однонаправлений лінійний список елементів абстрактного типу, що вводяться із клавіатури і виконує над ним операції: зчи¬ту¬вання та вивід інформації із списку; постановки елемента у список після заданого наперед елемента; виключення елемента із списку по заданому зна¬ченню. 
2. Сформувати однонаправлений лінійний список із елементів типу float, double, або класів, які містять інформаційне поля одного із вказаних типів, що вводяться із клавіатури. Знайти максимальний елемент списку і роздру¬ку¬вати його. Якщо числа не математичні, то треба створити  методику визначення максимального елемента
3. Сформувати однонаправлений лінійний список із елементів абстрактного типу, що вводяться із клавіатури. Відсортувати цей список по зростанню значень елементів. Якщо елементи не математичні числа, то треба створити  методику порівняння елементів.
4. Сформувати лінійний список із елементів абстрактного типу, отриманих за допомогою генератора випадкових величин абстрактного типу, рівномірно розподілених в інтервалі [0, 1]. Написати програму знаходження максимального елемента списку. Якщо елементи не математичні числа, то треба створити  методику порівняння на більше чи менше.
5. Заданий список із n чисел абстрактного типу. Написати програму, яка повертає значення true в тому випадку, якщо числа у списку розміщені в порядку зростання. Якщо елементи не математичні числа, то треба створити  методику сортування. 
6. Написати програму, яка із списку S чисел абстрактного типу формує список додатних елементів S1 та список від'ємних елементів S2. Якщо елементи не математичні сила, то треба створити  методику визначення додатності чи від’ємності.
7. Написати програму, яка об'єднує два впорядкованих по зростанню списки елементів абстрактного типу S1 та S2 в один впорядкований по зростанню список.
8. Написати програму, яка формує список S, включаючи в нього по одному разу елементи, які: а) входять хоча б в один із списків S1 або S2; б) входять одночасно в обидва списки S1 та S2; в) входять в список S1, але не входять до списку S2; 
9. Сформувати однонаправлений список елементів абстрактного типу та написати програму наступних операцій над ним: а) виключити із списку пов¬торні входження одного й того ж елемента; б) подвоїти кожен елемент списку.
10. Написати програму, яка в однонаправленому лінійному списку S елементів виконує наступні дії: а) заміняє перше входження списку S1 на список S2; б) витирає всі входження списку S1; в) робить копію списку S.
11. Заданий невпорядкований лінійний список елементів абстрактного типу. Роздрукувати у зворотному порядку всі числа, що знаходяться між міні¬маль¬ним та максимальним елементами. Якщо елементи не математичні, то треба створити  методики визначення порівняння та визначення максимального та мінімального значень.
13. Заданий список елементів абстрактного типу . Написати програму, яка: а) підраховує кількість елементів списку, у яких однакові сусідні елементи; б) витирає всі елементи, у яких однакові сусідні елементи;  Якщо елементи не математичні сила, то треба створити  методику порівняння на рівність.
14. Черга - це однонаправлений лінійний список, який поповнюється з кінця, а зменшується з початку. Написати програму: а) постановки елемента в чергу; б) виключення елемента з черги; в) перевірки, чи черга пуста.
15. Стек - це однонаправлений лінійний список, в якому елементи записуються і зчитуються з кінця списку. Написати програму, що виконує операції: а) включення елемента в стек; б) виключення елемента із стеку; 
16. Стек - це однонаправлений лінійний список, в якому елементи записуються і зчитуються з кінця списку. Написати програму, що виконує операції: а) очистки стеку; б) перевірки, чи стек пустий.
